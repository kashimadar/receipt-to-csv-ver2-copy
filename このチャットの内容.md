## 新しいフォルダをGitHubに追加する手順

### 1. GitHubでリポジトリを作成
1. https://github.com/new を開く
2. Repository name を入力
3. 「Create repository」をクリック

### 2. ターミナルで実行
```bash
cd "C:/Users/sakur/Desktop/claude作業用/github/フォルダ名"

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/kashimadar/リポジトリ名.git
git push -u origin main
```

※ SSL エラーが出たら `git -c http.sslVerify=false push` を使う
