# 現金領収書PDF → Excel 変換スキル

## 概要

スキャンPDFを画像として読み込み、AIビジョンで各領収書の内容を抽出し、Excelファイル（.xlsx）に出力する。

出力列：**日付 / 店名 / 金額 / 消費税8%**

---

## ステップ1: PDFを読み込む

ユーザーからPDFファイルのパスを受け取る。Readツールでそのパスを開く（スキャンPDFは画像として表示される）。

- 87ページ級の大きなPDFは全ページをいっきに読まず、まず最初の数ページで構造（1ページ＝1枚の領収書か、両面スキャンか等）を把握してから進める
- 両面スキャンの場合は重複ページを除外する（表と裏が別ページになっているケース多い）

---

## ステップ2: 各ページから情報を抽出する

各ページ（または各レシート）から以下の情報を読み取る：

| 列 | 内容 | 補足 |
|---|---|---|
| 日付 | `YYYY/MM/DD` 形式 | 和暦（令和7年＝2025年）も変換する |
| 店名 | 領収書に書かれた発行者・店舗名 | 手書きで読めない場合は「（手書き 店名不明）」と記入 |
| 金額 | 合計金額（数値のみ） | 税込金額を使う |
| 消費税8% | 軽減税率8%分の消費税額 | 記載がない・10%のみの場合は null |

**消費税8%について：**
- 8%（軽減税率）は主に「テイクアウトの飲食物」「食料品」に適用される
- SuicaチャージやPASMO等の交通系IC、郵便、電車・バス代は非課税または10%
- 領収書に「8%対象」「軽減税率」「※」などの明示があれば消費税8%欄に金額を記入
- 明示がなければ null（空欄）とする

---

## ステップ3: Excelファイルを生成する

### 環境確認

**Python + openpyxl** を使う。未インストールの場合は以下を実行：

```bash
pip install openpyxl
```

### スクリプトを使う方法

このスキルに同梱されているスクリプト `scripts/create_excel.py` を使う。
スキルのディレクトリパスはこのSKILL.mdと同じ場所にある。

```bash
python "<スキルディレクトリ>/scripts/create_excel.py" '<JSON>' '<出力パス>'
```

JSONの形式：
```json
[
  {"date": "2025/05/03", "store": "JR東日本（土気駅）Suicaチャージ", "amount": 10000, "tax8": null},
  {"date": "2025/05/11", "store": "銀座コージーコーナー日暮里店", "amount": 756, "tax8": 56}
]
```

### 出力パスの注意

- 日本語ファイル名はOK
- 出力先はユーザーの指定したパスを使う

---

## ステップ4: 完了を報告する

- 保存先のフルパスをユーザーに伝える
- 処理した領収書枚数と、8%税額が見つかった件数を報告する
- 金額の合計も一言添えると親切

---

## よくあるケース

### 試し（10枚程度）の場合
- まず最初の10ページ前後を読んで10件分を抽出してExcelに出力する
- 「うまくいったら全部やる」という確認を得てから全件処理する

### 全件処理の場合
- 重複除外後の枚数を確認してからExcelに一括出力する
- 大量のページは20〜30ページずつ読んでデータを積み上げていく

---

## 出力ファイル名の例

```
<氏名>_現金領収書一覧.xlsx
<氏名>_現金領収書一覧_試作.xlsx
現金領収書一覧_2025.xlsx
```

ユーザーが名前を指定した場合はそれを優先する。指定がなければ `<氏名>_現金領収書一覧.xlsx` の形式を使う。
